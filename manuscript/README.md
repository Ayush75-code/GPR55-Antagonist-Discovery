# Manuscript: GPR55 Antagonist Discovery

This folder contains materials for the research publication.

## Target Journal
**Elsevier** (e.g., *European Journal of Medicinal Chemistry*, *Bioorganic & Medicinal Chemistry*, or *Computational and Structural Biotechnology Journal*)

## Structure
```
manuscript/
├── figures/          # Publication-ready figures (PyMOL renders, graphs)
├── supplementary/    # Supplementary data (tables, methods)
└── README.md
```

## What's Tracked by Git
- `figures/*.png`, `*.tiff`, `*.svg` (publication figures)
- `supplementary/*.pdf`, `*.csv` (supplementary materials)

## What's NOT Tracked
- `drafts/` folder (private until publication)
- `*.docx`, `*.doc` (Word documents - use LaTeX or share privately)

## Elsevier Submission Checklist
- [ ] Abstract (structured, 250 words max)
- [ ] Keywords (5-6)
- [ ] Graphical Abstract (530×300 px)
- [ ] Highlights (3-5 bullet points, 85 chars each)
- [ ] ORCID for all authors
- [ ] Data availability statement
- [ ] Cover letter

## Figure Requirements (Elsevier)
- Minimum 300 DPI
- Formats: TIFF, EPS, PDF, PNG
- Max width: 190 mm (two-column)
